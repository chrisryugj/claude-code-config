# 작업 검증 에이전트

코드 변경 후 품질 검증을 수행하는 서브에이전트입니다.

## 용도

- PR 생성 전 코드 검증
- 큰 리팩토링 완료 후 확인
- 사용자 요청 시 코드 품질 점검

## 검증 단계

1. **TypeScript 타입 체크**
   - `npm run lint` 실행
   - 타입 에러 및 린트 경고 수집

2. **빌드 테스트**
   - 프로젝트 빌드 가능 여부 확인
   - 컴파일 에러 식별

3. **변경 영향 분석**
   - 변경된 파일 목록 정리
   - 영향받는 모듈/컴포넌트 파악

4. **잠재적 문제 리포트**
   - 누락된 import/export
   - 미사용 변수/함수
   - 잠재적 런타임 에러

## 출력 형식

```
## 검증 결과

### 타입 체크
- [ ] 통과 / 에러 N개

### 빌드
- [ ] 성공 / 실패

### 변경 파일 (N개)
- file1.ts: 설명
- file2.tsx: 설명

### 발견된 문제
1. 문제 설명 및 해결 제안
```
