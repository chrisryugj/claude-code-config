# Memory Bank 초기화

현재 프로젝트에 Memory Bank 시스템을 설정합니다.

## 실행할 작업

1. **폴더 구조 생성**
   - `.claude/memory/` 폴더 생성

2. **기본 파일 생성**
   - `activeContext.md` - 현재 작업 상태
   - `decisions.md` - 아키텍처 결정
   - `patterns.md` - 코드 패턴

3. **gitignore 업데이트** (선택적)
   - `.claude/memory/` 추가 여부 확인

4. **초기 분석**
   - 프로젝트 구조 파악
   - 기존 패턴 식별
   - Memory Bank에 초기 정보 기록
