# Memory Bank 동기화

현재 세션의 작업 내용을 Memory Bank에 저장합니다.

## 실행할 작업

1. **현재 상태 수집**
   - 진행 중인 작업 확인
   - 완료된 작업 정리
   - 발생한 문제/해결책 정리

2. **activeContext.md 업데이트**
   - 현재 작업 상태 반영
   - 최근 변경 내용 추가
   - 다음 세션 할 일 정리

3. **decisions.md 업데이트** (해당 시)
   - 새로운 아키텍처 결정 기록
   - 기술 선택 이유 문서화

4. **patterns.md 업데이트** (해당 시)
   - 새로 발견한 패턴 기록
   - 코드 컨벤션 업데이트

## 업데이트 원칙

- **간결하게**: 핵심 정보만 기록
- **구체적으로**: 파일명, 함수명 명시
- **이유 포함**: 왜 그렇게 했는지 기록
- **중복 제거**: 기존 내용과 중복 피하기

## 자동 호출 권장 시점

- 큰 기능 구현 완료 후
- 중요한 버그 수정 후
- 세션 종료 전
- 아키텍처 결정 후
