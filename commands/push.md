# Git Commit & Push

변경사항을 커밋하고 원격 저장소에 푸시합니다.

## 실행할 작업
1. `git status`로 변경된 파일 확인
2. `git diff`로 변경 내용 확인
3. `git log -3 --oneline`로 최근 커밋 스타일 확인
4. 변경사항 분석 후 적절한 커밋 메시지 작성
5. `git add .`로 모든 변경사항 스테이징
6. `git commit`으로 커밋
7. `git push`로 원격에 푸시
8. 완료 후 결과 요약

## 커밋 메시지 형식
- 한글로 작성
- prefix 사용: feat, fix, refactor, docs, style, chore
- 예: `feat: 새로운 테마 추가`
